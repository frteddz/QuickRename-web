# QuickRename

> Bulk file renamer — rename dozens of files in seconds with powerful patterns.

## Features

- Add prefixes and suffixes to file names
- Find-and-replace text across multiple file names
- Insert sequential numbering with custom start and padding
- Change file extensions in bulk
- Remove or replace specific characters
- Convert between camelCase, snake_case, and kebab-case
- Apply date and time stamps to file names
- Preview changes before committing the rename
- Undo last rename operation

## Why QuickRename?

QuickRename runs entirely client-side — your files stay on your computer. There's no upload step, no server processing, and no file size limits. It's free, instant, and built to handle renaming workflows that would otherwise require installing a native app or writing complex shell scripts.
